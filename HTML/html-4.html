In HTML, there are a variety of tags, each serving a unique purpose in creating and structuring content. Here's a categorized list of the most commonly used HTML tags: --- ### 1. **Document Structure Tags** - ``: Root element of an HTML document. - ``: Defines the title of the document, shown in the browser tab. - `<body>`: Contains the content of the HTML document. ### 2. **Metadata Tags** - `<meta>`: Specifies metadata such as character set, author, description, and viewport. - `<link>`: Used to link to external resources like CSS files. - <style></style>: Contains internal CSS. - <script></script>: Embeds or links to JavaScript. ### 3. **Text Content Tags** - `<h1>` to `<h6>`: Headings, with `<h1>` as the highest level and `<h6>` as the lowest. - `<p>`: Paragraph. - `<br>`: Line break. - `<hr>`: Horizontal rule, used to create a thematic break. - `<pre>`: Preserves whitespace and displays text in a fixed-width font. ### 4. **Formatting Tags** - `<b>`: Bold text. - `<i>`: Italic text. - `<u>`: Underlined text. - `<strong>`: Important text, usually bold. - `<em>`: Emphasized text, usually italic. - `<small>`: Small-sized text. - `<mark>`: Highlighted text. - `<sub>` and `<sup>`: Subscript and superscript text. - `<ins>`: Inserted (underlined) text, often used for additions. - `<del>`: Deleted (strikethrough) text, often used for removals. ### 5. **Grouping Content Tags** - `<div>`: Block-level container, typically used for layout and structure. - `<span>`: Inline container, used to style small sections within a line. - `<header>`: Defines a header section. - `<footer>`: Defines a footer section. - `<main>`: Main content of the document. - `<section>`: A thematic grouping of content. - `<article>`: Self-contained content, such as a blog post. - `<aside>`: Content related to the main content, often used for sidebars. - `<nav>`: Navigation links. ### 6. **List Tags** - `<ul>`: Unordered list (bulleted). <ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> <li>Java</li> <li>Python</li> </ul> - `<ol>`: Ordered list (numbered). <ol> <li>Update your resume</li> <li>Apply to job portals</li> <li>Prepare for coding interviews</li> <li>Attend technical interviews</li> <li>Receive your offer letter</li> </ol> - `<li>`: List item. - `<dl>`: Description list. <dl> <dt>HTML</dt> <dd>HyperText Markup Language, used to create the structure of web pages.</dd> <dt>CSS</dt> <dd>Cascading Style Sheets, used for styling the appearance of web pages.</dd> <dt>JavaScript</dt> <dd>A programming language that adds interactivity to websites.</dd> </dl> - `<dt>`: Description term. - `<dd>`: Description definition. You can nest one list inside another to create more complex structures. This is useful when you want to display hierarchical data. Code Example: <ul> <li>Frontend Technologies <ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul> </li> <li>Backend Technologies <ul> <li>Java</li> <li>Python</li> <li>Node.js</li> </ul> </li> </ul> ### 7. **Table Tags** - `<table>`: Table element. - `<tr>`: Table row. - `<td>`: Table data cell. - `<th>`: Table header cell. - `<thead>`, `<tbody>`, `<tfoot>`: Grouping elements for table sections. ### 8. **Form Tags** - `<form>`: Form container. - `<input>`: Various types of input fields (text, password, checkbox, etc.). - `<textarea>`: Multi-line text input. - `<button>`: Clickable button. - `<select>`: Dropdown list. - `<option>`: Options within a `<select>`. - `<label>`: Labels for form elements. - `<fieldset>` and `<legend>`: Grouping of related form elements. ### 9. **Media Tags** - `<img>`: Embeds images. - `<audio>`: Embeds audio files. <h1>Listen to the Audio</h1> <audio src="https://example.com/audio-file.mp3" controls loop autoplay muted> Your browser does not support the audio tag. </audio> - `<source>`: Defines multiple media resources for `<audio>` or `<video>`. - `<track>`: Adds text tracks, like captions, to `<video>` or `<audio>`. ### 10. **Embedded Content Tags** - `<iframe>`: Embeds another HTML page within the current page. <h1>Iframe with srcdoc Example</h1> <iframe src="https://www.kodnest.com/" height="300px" width="500px" frameborder="0" scrolling="yes"> </iframe> *************** <!-- <h1>Iframe with srcdoc Example</h1> <iframe srcdoc=' <a href="https://www.kodnest.com/" target="iframe1"> Click here to open Kodnest website </a>' height="300" width="500" name="iframe1"> </iframe> --> - `<embed>`: Embeds external content, such as a plugin. - `<object>`: Represents external resources, like multimedia or interactive content. - `<param>`: Defines parameters for `<object>`. ### 11. **Scripting Tags** - `<script>`: Embeds JavaScript. - `<noscript>`: Provides content if scripting is unsupported or disabled. ### 12. **Interactive Tags** - `<details>`: Disclosure widget for showing/hiding additional information. - `<summary>`: Summary for the `<details>` element. - `<dialog>`: Represents a dialog box or other interactive component. ### 13. **Semantic Tags (HTML5)** - `<article>`, `<aside>`, `<figure>`, `<figcaption>`, `<footer>`, `<header>`, `<main>`, `<mark>`, `<nav>`, `<section>`, `<time>`: Used to provide better semantic structure and meaning. ### 14. **Deprecated Tags** (Generally Avoided in Modern HTML) - `<font>`: Used to style font (now replaced by CSS). - `<center>`: Centers content (now replaced by CSS). - `<big>`, `<strike>`, `<tt>`: Outdated tags for formatting and typography.
